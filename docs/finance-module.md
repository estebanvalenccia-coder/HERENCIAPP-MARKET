# Modulo Finanzas Herencia Market

Modulo para ventas diarias, gastos, cierre de caja, resumen mensual y graficos financieros.

## Rutas propuestas

- /admin/finanzas
- /admin/caja
- /admin/reportes

## Funciones

- Registrar ventas del dia
- Registrar gastos
- Cerrar caja diaria
- Ver resumen mensual
- Ver graficos de ingresos, gastos y beneficios
